# LOF Monitor - 海外LOF溢价监控系统

一个自动化监控海外上市LOF（上市开放式基金）溢价情况的工具，帮助你发现套利机会。

## 📊 功能特性

- **实时监控全球主要市场** - 美股、港股、大宗商品、REITs等多个资产类别的LOF基金
- **自动计算溢价率** - 基于海外资产实时价格和T-1日官方净值计算IOPV与溢价率
- **申购状态识别** - 实时爬取官方网站，识别基金的申购/限购状态
- **汇率动态调整** - 自动获取人民币汇率变化，准确计算含汇兑损益的IOPV
- **智能推送通知** - 发现高溢价机会（>3%）且可申购的基金时，自动通过PushDeer推送提醒
- **定时自动运行** - 通过GitHub Actions每个交易日两次（北京时间13:30、14:30）自动执行

## 🚀 快速开始

### 前置条件

- GitHub 账户
- PushDeer 账户（用于接收推送通知）
- Python 3.10+（如果在本地运行）

### 部署步骤

#### 1. Fork/Clone 本仓库
```bash
git clone https://github.com/CrazyExploration/LOF-monitor.git
cd LOF-monitor
```

#### 2. 配置 GitHub Secrets

在仓库 Settings → Secrets and variables → Actions 中添加：

| Secret 名称 | 说明 |
|-----------|------|
| `PUSH_KEY` | 你的 PushDeer PUSH_KEY |

获取方式：
1. 登录 [PushDeer](https://www.pushdeer.com)
2. 获取你的 PUSH_KEY（通常是一长串随机字符）

#### 3. 启用 GitHub Actions

确保仓库设置中 Actions 已启用：
- Settings → Actions → General
- 选择 "Allow all actions and reusable workflows"

#### 4. 改仓库为公有（重要！）

GitHub 对私有仓库的计划任务支持不佳。建议改为公有：
- Settings → General → Danger Zone → Change visibility → Public

#### 5. 完成！

工作流将在每个交易日 13:30 和 14:30（北京时间）自动运行。

### 本地运行

如果想在本地测试：

```bash
# 安装依赖
pip install yfinance pandas requests

# 设置环境变量
export PUSH_KEY="your_pushdeer_key_here"

# 运行监控脚本
python monitor.py
```

## 📈 工作原理

### 数据获取流程

```
┌─────────────────────────────────────┐
│  获取海外市场实时数据 (yfinance)   │ 美股、港股、商品期货
└────────────┬────────────────────────┘
             ↓
┌─────────────────────────────────────┐
│  获取A股LOF场内实时价格             │ 新浪财经API
│  获取T-1日官方净值                  │ 天天基金API
└────────────┬────────────────────────┘
             ↓
┌─────────────────────────────────────┐
│  爬取申购状态                       │ 东方财富官网
│  识别 暂停/限购/自由申购 状态       │ 正则匹配
└────────────┬────────────────────────┘
             ↓
┌─────────────────────────────────────┐
│  计算实时 IOPV（理论净值）          │
│  IOPV = T-1净值 × (1+海外涨幅)     │
│        × (1+汇率变化)               │
└────────────┬────────────────────────┘
             ↓
┌─────────────────────────────────────┐
│  计算溢价率                         │
│  溢价率 = (现价/IOPV) - 1           │
└────────────┬────────────────────────┘
             ↓
┌─────────────────────────────────────┐
│  判断套利机会                       │
│  IF 溢价率>3% AND 可申购            │
│  THEN 发送 PushDeer 推送            │
└─────────────────────────────────────┘
```

### IOPV 计算公式

**含汇兑的基金**（美股、港股等）：
```
IOPV = T-1净值 × (1 + 海外资产涨幅%) × (1 + 人民币贬值%)
```

**不含汇兑的基金**（以美元计价）：
```
IOPV = T-1净值 × (1 + 海外资产涨幅%)
```

## ⚙️ 配置说明

### Cron 时间表

当前配置在每个交易日（周一至周五）执行两次：

| 时间 | 用途 | 说明 |
|------|------|------|
| 13:30 | 早盘机会 | 上午交易收盘后，及时捕捉上午溢价 |
| 14:30 | 晚盘机会 | 下午开盘半小时，发现晚盘新机会 |

时间为北京时间。如需修改，编辑 `.github/workflows/lof_cron.yml` 中的 cron 表达式。

**Cron 转换参考**：
- Cron 使用 UTC 时间
- 北京时间 = UTC + 8 小时
- 例：北京时间 13:30 = UTC 05:30 = `30 5 * * 1-5`

## 🐛 常见问题

**Q: 为什么计划任务没有自动运行？**
- A: 私有仓库的计划任务支持不好。需要改为公有仓库。

**Q: 推送没有收到怎么办？**
- A: 检查以下几点：
  1. 确保 PUSH_KEY 正确且有效
  2. 检查 PushDeer 应用设置
  3. 查看 GitHub Actions 运行日志排查错误

**Q: 怎么修改监控的基金品种？**
- A: 编辑 `monitor.py` 中的 `FUND_MAP` 字典，按照现有格式添加/删除基金代码。

**Q: 溢价率阈值能否调整？**
- A: 可以，在 `monitor.py` 中修改条件，将 `0.03` 改为你需要的阈值（如 `0.05` 表示5%）。
